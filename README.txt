
Este es un prototipo de la app INFONA construida con Python y Streamlit.

Instrucciones para ejecutar localmente:
1. Instala Streamlit si no lo tienes:
   pip install streamlit

2. Ejecuta la aplicación:
   streamlit run app.py

Esto abrirá una interfaz web en tu navegador donde podrás usar el simulador, formulario y chatbot simulado.
